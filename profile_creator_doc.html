Profile Creator

Step-by-step guide to generate MT5 chart profiles from your setfiles.

1
Load and edit your setfiles, then choose the Profile Creator

Open the tool at fxlive.trading/metatrader-setfile-editor.html or use the local .html file directly. Once your setfiles are loaded and edited in the MT5 Set Editor tab, switch to the Profile Creator tab.

Enter the symbol as it is used by your broker

Enter the symbol exactly as your broker displays it (e.g. XAUUSD-ECN). Then configure the timeframe using period_type and period_size.

3
Expert information are already prefilled

The EA name, path and expertmode are automatically populated from your setfile. Verify that the path matches your broker's MT5 installation.

EXPERT

4
Don't forget to check if all the other parameters are correct

Review every parameter in the inputs section before saving. Use Show differences only to quickly spot any values that differ from your reference setfile.

5
Click on the « Save All .chr »

Once everything is verified, click Save All .chr. A zip file is generated containing all your profile chart files.

📦 You obtain a zip file with all the profile .chr files in it.
6
Copy those files in a specific folder in the /MQL5/Profiles/Charts

Extract the zip and place all .chr files inside a dedicated subfolder in your MT5 data directory.

7
If it's a new folder, you have to restart MT5 then load the profile in MT5

MT5 must be restarted once for new profile folders to appear in the list. Then navigate to File → Profiles and select your profile.

8
All charts with working instances of UBS with specific setfiles are loaded and ready to trade

MT5 opens all charts simultaneously, each running its own EA instance with its dedicated setfile configuration.

✦ Bonus

Quick profile switching

You can switch to the Default profile to close all chart windows in one click.

Then reload your custom profile after any setfile modification to quickly test your changes without manually reopening each chart.
